# tetrika_test

**Требования:**  
- Python 3.9 или выше  
- pip
## 1. Клонировать репозиторий
```bash
git clone https://github.com/StatusSSS/tetrika_test.git

cd tetrika_test
```

## 2. Установить библиотеки

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```


## 3. Запуск тестов
```bash
pytest -q
```
